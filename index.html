Niryo NED2 RFID Sorting

Robotics · Automation

Overview

Integrated Niryo NED2 6-axis robotic arm with TurtleBot4 mobile platform for automated lab sorting. ESP32 + PN532 reads RFID tags to identify each test tube and route it to the correct bin.

Key Achievements

  • Resolved ±15mm offset by removing calibrate_auto() and implementing manual joint-space calibration
  • Flask middleware orchestrates ESP32 (WebSocket), Niryo (TCP PyNiryo), TurtleBot4 (ROS2)
  • Non-blocking WebSocket stream for continuous RFID scanning during arm movement
  • Ubuntu 24.04, ROS2 Jazzy; ESP32 + PN532 over I2C · Robot IP: 192.168.8.150

Gesture Multi-Drone Control

AI · Computer Vision · Robotics

Overview

Two-hand gesture control of 3 simulated DJI Mavic2Pro drones in Webots. Left hand selects which drone(s); right hand commands actions. Custom 5-layer neural network built from scratch in NumPy.

Key Achievements

  • Left hand: 4 classes (Drone 1/2/3/All) — Right hand: 6 actions (Takeoff/Land/Up/Down/Flip/Hover)
  • Custom fully-connected network: 35→64→32→16→N trained with Adam, L-BFGS, and Nelder-Mead
  • Mass compensation mid-flight — drone adjusts thrust proportionally when mass changes
  • 35 hand-landmark features per hand extracted from MediaPipe HandLandmarker API

TurtleBot4 Queuing Navigation

ROS2 · Autonomous Nav

Overview

Dynamic occupancy-aware navigation for TurtleBot4. The robot automatically reroutes to a queue position when a target waypoint is occupied. Built on ROS2 Jazzy Nav2 with SLAM.

Key Achievements

  • Server node maintains waypoint occupancy states (FREE/OCCUPIED); client reroutes automatically
  • Nav2 action server for sequential multi-waypoint missions with dynamic obstacle avoidance
  • Precision docking using ArUco markers at ±5cm; complete 40m² SLAM map of H101 lab

Camera Servo Tracking (RPi)

Computer Vision · Embedded

Overview

Real-time multi-object detection and tracking on Raspberry Pi 4 using YOLOv8 (NCNN-exported) with two MG996R servos for pan-tilt camera control.

Key Achievements

  • Picamera2 + YOLOv8n NCNN model for low-latency inference on Pi hardware
  • PID-style proportional control (pan + tilt) keeps selected class centred in frame
  • Configurable target class (default: person), deadband, max step angle, and proportional gain
  • External 5V servo power supply; pigpio for PWM signal generation on GPIO17/18

Autonomous Mapping Robot

Embedded · Control Systems

Overview

Two-Arduino autonomous robot that maps a room by moving forward until hitting a wall, then returns with 5 equally-spaced stops. Bridge slowdown and obstacle detection included.

Key Achievements

  • Master board: 3 HC-SR04 sensors + LEDs; Slave board: DRV8833 motor PWM + encoder counting
  • 850 encoder ticks = 1 wheel revolution; minimum detectable distance 7.06μm
  • Bridge upward sensor triggers 50% speed slowdown; rear obstacle terminates run safely
  • Max velocity 36.63 cm/s at PWM 200; one-way serial Master Pin11 → Slave Pin8
